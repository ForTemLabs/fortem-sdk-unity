# ForTem Core - Getting Started Sample

This sample demonstrates basic usage of the ForTem Core framework.

## What's Included

- Example script showing framework initialization
- Basic setup for using ForTem Core in your project

## How to Use

1. This sample will be extracted to your Assets folder when imported from Package Manager
2. Review the example scripts to understand the framework structure
3. Adapt the examples to your project needs

## Next Steps

- Explore the Runtime folder to see available core utilities
- Check the Documentation for detailed API reference
- Create your own extensions based on the framework structure
